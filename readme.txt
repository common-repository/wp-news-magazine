== Description ==
WP News is a WordPress News / Magazine Plugin. You can create any news site easily. 
This plugin includes Elementor Addon, WordPress Widget, Custom styles options with different customization option, News Slider, News Grid, News Tab.

== Installation ==
This section describes how to install the WP Instagram Plugins for WordPress get it working.

= 1) Install =

1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WP News Magazine".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `wp-news-magazine` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. Go to the WP News Settings Options.
2. Add the Google Map Api (If Google map is required).
